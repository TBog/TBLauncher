Motivos para desenvolver meu próprio launcher:
🔹Limpe a tela principal para aproveitar o papel de parede
🔸Acesso rápido a aplicativos pesquisando
🔹Compatível com pacote de ícones
🔸Capacidade de personalizar cores e comportamentos

Novas características:
⦾ Atualize a lista de aplicativos quando forem detectadas alterações
⦿ Formas de ícones adaptáveis
⦾ Ícones de cache para exibição rápida de resultados
⦿ Tags para organizar melhor os apps
⦾ Encaixar itens para filtrar resultados
⦿ Pré-visualização de cores de preferência
⦾ Atalhos para aplicativos, contatos, etc.
